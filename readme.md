EnergiAI es una solución inteligente que analiza los patrones de consumo de energía eléctrica de viviendas y pequeños establecimientos, los clasifica según su perfil de eficiencia y genera recomendaciones accionables — junto con una estimación financiera del impacto — para apoyar decisiones más conscientes y sostenibles.

El proyecto combina un pipeline de **Ciencia de Datos** con una **API REST** construida en Java 25 + Spring 4.1.0, y se desplegará sobre servicios de **Oracle Cloud Infrastructure (OCI)**.
